AUTO FILE SORTER FOR YOUR SITE

HOW TO USE:
1. Unzip this into your main site folder (where all your HTML, CSS, JS, images are).
2. Make sure Python is installed on your computer.
3. Open terminal or command prompt.
4. Run:
   python auto_file_sorter.py

WHAT IT DOES:
- Moves all files into proper folders:
  - HTML → /pages/
  - CSS → /styles/
  - JS → /scripts/
  - Images → /assets/images/
  - Fonts → /assets/fonts/
  - Videos → /assets/videos/
  - Audio → /assets/audio/
